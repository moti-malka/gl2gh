## Interactive Elements

### Checkbox Behavior
- Click checkbox to toggle selection
- Click anywhere on project name to toggle
- Selected projects get blue highlight

### Target Repo Input
- Editable text input with monospace font
- Disabled (grayed out) when project not selected
- Default value auto-generated from path_with_namespace
- Format: `owner/repo-name`

### Action Buttons

**Select All**
- Checks all project checkboxes
- Enables all target repo inputs

**Deselect All**
- Unchecks all project checkboxes
- Disables all target repo inputs

**Back to Discovery**
- Optional - can be hidden
- Returns to discovery configuration
- Doesn't save changes

**Continue to Export**
- Disabled when no projects selected
- Saves selection to backend
- Proceeds to next migration stage
- Shows success toast notification

## Responsive Design

### Desktop (>768px)
- Full width cards
- Metrics in horizontal row
- Side-by-side buttons

### Mobile (<768px)
- Stacked layout
- Metrics in vertical list
- Full-width buttons
- Smaller text sizes

## State Indicators

### Loading States
1. Initial load: Full page spinner
2. Loading discovery: Spinner in panel area
3. Saving selection: Button shows loading
